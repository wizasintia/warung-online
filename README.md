# Sistem Informasi Warung Online

Project ini merupakan tugas Ujian Akhir Semester
mata kuliah Pemrograman Berbasis Web Lanjutan (PBWL).

Aplikasi dibangun menggunakan:
- Laravel 12.x
- Bootstrap
- MySQL

## Fitur Sistem
- Login dan Logout
- CRUD Data Produk
- CRUD Data Pesanan
- Manajemen Pengguna
- Tampilan Responsif dengan Bootstrap

## Cara Menjalankan Project
1. composer install
2. npm install
3. cp .env.example .env
4. php artisan key:generate
5. Atur database di file .env
6. php artisan migrate
7. npm run dev
8. php artisan serve
